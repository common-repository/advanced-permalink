== Description ==

Struggling with SEO friendly links ? need to redefine your permalinks ? Here is the solution. For any further customization feel free to raise a ticket on our [website](http://www.mingocommerce.com/contact-us/)

== Installation ==

Manual

1. Upload the plugin folder to the '/wp-content/plugins/' directory .

2. Activate the plugin through the 'Plugins' menu in WordPress.

Automatic

1. Go to plugin list from 'dashboard'. Search our plugin and click to install

2. Activate it.
